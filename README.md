# Sistema de Compra de Passagem

Projeto desenvolvido em Java para simular uma interface de compra de passagem, utilizando JavaFX, Maven e Scene Builder.

## Sobre o projeto

Este projeto foi desenvolvido durante a aula de **CAAV** com o objetivo de praticar a criação de interfaces gráficas em Java. A aplicação simula uma tela de compra de passagem, permitindo ao usuário interagir com campos e opções relacionados ao processo de aquisição de uma passagem.

A interface foi criada com o **Scene Builder**, facilitando a construção visual das telas e a organização dos componentes gráficos da aplicação.

## Funcionalidades

- Interface gráfica para compra de passagem
- Campos para preenchimento de informações da viagem
- Seleção de opções relacionadas à passagem
- Organização visual criada com Scene Builder
- Estrutura do projeto gerenciada com Maven
- Execução como aplicação JavaFX

## Tecnologias utilizadas

- Java
- JavaFX
- Maven
- Scene Builder
- FXML
- Eclipse IDE ou IntelliJ IDEA

## Objetivo acadêmico

O objetivo do projeto é aplicar conceitos de desenvolvimento de interfaces gráficas com JavaFX, utilizando Scene Builder para criação das telas e Maven para gerenciamento da estrutura e dependências do projeto.

## Interface
A interface da compra de passagem foi construída com o Scene Builder, utilizando arquivos FXML para separar a parte visual da lógica da aplicação.
